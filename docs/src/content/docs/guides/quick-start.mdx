## Get a project

Open an existing JVM application project using Gradle, or create a new one.

<Tabs syncKey="os">
<TabItem label="Windows">

```shell title="Creating a new project"
gradle init
```

</TabItem>
<TabItem label="Linux">

```shell title="Creating a new project"
gradle init
```

</TabItem>
</Tabs>

## Add the plugin

<Tabs syncKey="dsl">
<TabItem label="Kotlin">

```kts
// build.gradle.kts
plugins {
    application
    id("de.infolektuell.jpackage") version "x.y.z"
}

repositories {
    mavenCentral()
}

java {
    toolchain {
        languageVersion = JavaLanguageVersion.of(25)
    }
}

application {
    applicationName = "jTunes"
    mainClass = "org.example.App"
}
```

</TabItem>
<TabItem label="Groovy">

```groovy
// build.gradle
plugins {
    id application
    id 'de.infolektuell.jpackage' version 'x.y.z'
}

repositories {
    mavenCentral()
}

java {
    toolchain {
        languageVersion = JavaLanguageVersion.of 25
    }
}

application {
    applicationName = 'jTunes'
    mainClass = 'org.example.App'
}
```

</TabItem>
</Tabs>

##  Inspect the tasks

The plugin uses the `build/jpackage` directory for its outputs and adds its tasks to the Application group:

<Tabs syncKey="os">
<TabItem label="Windows">

```shell title="Listing the tasks"
gradlew.bat tasks
```

</TabItem>
<TabItem label="Linux">

```shell title="Listing the tasks"
./gradlew tasks
```

</TabItem>
</Tabs>

- jdeps
- jlink
- appImage
- appInstaller

:::tip
Probably the project needs some additional configuration.
The following guides cover different setups and use-cases.
:::
